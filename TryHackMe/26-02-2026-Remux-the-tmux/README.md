Important commands and actions practiced
- Starting tmux with: tmux
- Listing sessions: tmux ls
- Creating a new session: tmux new -s session_name
- Attaching to a session: tmux attach -t session_name
- Detaching from a session: prefix + d
- Killing a session: tmux kill-session -t session_name
- Splitting panes horizontally and vertically using prefix commands
- Switching between panes using arrow keys after prefix
- Creating and switching windows using prefix commands
- Using copy mode to scroll and copy text inside tmux
- Swapping panes to reorganize the layout
- Using and testing a custom .tmux.conf configuration
