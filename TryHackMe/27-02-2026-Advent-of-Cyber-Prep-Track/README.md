Overview
- Advent of Cyber Prep Track is a beginner friendly training room created to prepare users for the Advent of Cyber event
- The room contains multiple mini challenges that teach basic cyber security concepts
- These tasks help beginners learn how to investigate systems, analyze files, and understand common security problems
- The story is set in a town called Wareville where something suspicious is happening during the SOCMAS celebration
- Participants complete challenges to investigate security incidents and learn practical skills

Lab Story
- Wareville is preparing for the SOCMAS cyber festival
- Systems start behaving strangely and suspicious activities appear
- The team investigates multiple small incidents such as malware files, data leaks, and suspicious log activity
- Each challenge teaches an important cyber security concept
- By completing all challenges the system is secured and the investigation is solved

Challenge 1 - Password Pandemonium
- The first task introduces basic password concepts
- The goal is to understand password strength and authentication basics
- Flag obtained
THM{StrongStart}

Challenge 2 - The Suspicious Chocolate.exe
- A suspicious executable file is discovered on a USB drive
- The file is analyzed using VirusTotal
- The scan reveals malware detection results
- Flag obtained
THM{NotSoSweet}

Challenge 3 - Welcome to the AttackBox
- The TryHackMe AttackBox environment is used
- Basic Linux commands are executed to explore directories
- A hidden welcome file is opened
- Flag obtained
THM{Ready2Hack}

Challenge 4 - The CMD Conundrum
- A Windows system contains hidden files
- Windows command prompt is used to explore directories
- Hidden files are discovered inside a folder
- Flag obtained
THM{WhereIsMcSkidy}

Challenge 5 - Linux Lore
- Linux commands are used to investigate a user's home directory
- Hidden files are revealed using ls -la
- A secret message file is opened
- Flag obtained
THM{TrustNoBunny}

Challenge 6 - The Leak in the List
- A possible data breach is investigated
- The email address mcskidy@tbfc.com is checked in a simulated breach database
- A compromised service is discovered
- Flag obtained
THM{LeakedAndFound}

Challenge 7 - WiFi Woes in Wareville
- A router is discovered using default credentials
- The device is accessed through the web interface
- The weak password is replaced with a stronger configuration
- Flag obtained
THM{NoMoreDefault}

Challenge 8 - The App Trap
- A suspicious third party application is connected to an account
- The application has excessive permissions
- Access is revoked to secure the account
- Flag obtained
THM{AppTrapped}

Challenge 9 - The Chatbot Confession
- A chatbot conversation is analyzed
- Some messages reveal sensitive information such as URLs and credentials
- These messages are identified and flagged
- Flag obtained
THM{DontFeedTheBot}

Challenge 10 - The Bunny's Browser Trail
- Web server logs are analyzed
- Different user agents are compared
- A suspicious automated user agent called BunnyOS is discovered
- Flag obtained
THM{EastmasIsComing}
