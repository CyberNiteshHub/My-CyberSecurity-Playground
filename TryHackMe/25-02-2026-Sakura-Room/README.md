Overview
- This room is based on OSINT techniques and investigations
- The main goal is to identify a person and trace their digital footprint using public information
- No exploitation or hacking tools are required in this lab
- The lab focuses on thinking, searching, analyzing, and connecting small clues
- All tasks were completed inside the TryHackMe environment

Story and Objective
- We are given small clues about a person named Aiko
- The task is to use OSINT methods to find usernames, profiles, locations, and related information
- The investigation is done using social media, search engines, image search, GitHub, and other public sources
- Each step reveals more information about the target
- Finally, we connect all the dots to complete the room
